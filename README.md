# Hybrid Feature-Based Prediction of Suicide-Related Activity on Twitter

## Project Overview

Hybrid Feature-Based Prediction of Suicide-Related Activity on Twitter is a web application developed using **Python, Django, Machine Learning, and Natural Language Processing (NLP)**. The primary objective of this project is to identify suicide-related content from Twitter posts by analyzing textual and sentiment-based features. The system helps in the early detection of high-risk social media content and supports timely intervention.

## Objectives

* Detect suicide-related tweets from Twitter data.
* Perform sentiment analysis on tweet content.
* Improve prediction accuracy using hybrid feature extraction.
* Provide data visualization and analytical reports.

## Features

* User Registration and Login
* Admin Dashboard
* Tweet Dataset Upload (Excel)
* Tweet Search
* Sentiment Classification
* Suicide-Related Tweet Prediction
* Positive and Negative Tweet Analysis
* Trending Topic Analysis
* Data Visualization with Charts
* Prediction Accuracy Report

## Workflow

1. Upload the Twitter dataset.
2. Preprocess and clean the tweet data.
3. Extract hybrid features from the tweets.
4. Apply machine learning algorithms for classification.
5. Predict whether tweets are suicide-related or non-suicide-related.
6. Display sentiment analysis, prediction results, and visualization charts.

## Installation

```bash
git clone https://github.com/your-username/Hybrid-Feature-Prediction-of-Suicide-Related-Activity-on-Twitter.git
cd Hybrid-Feature-Prediction-of-Suicide-Related-Activity-on-Twitter
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
